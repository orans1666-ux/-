# Sixth Stars Rebuild

نسخة جديدة مستقلة من موقع صالون النجمة السادسة، مبنية بأكواد جديدة داخل مجلد `Sixth_Stars_Rebuild`.

## التشغيل

افتح `index.html` مباشرة، أو استخدم الرابط المحلي الذي يتم تشغيله عبر خادم بسيط.

## المزايا

- HTML5 و CSS3 و JavaScript فقط.
- تصميم جديد مختلف عن النسخة السابقة.
- هيدر Sticky متجاوب.
- دعم العربية والإنجليزية مع RTL/LTR.
- دعم Light/Dark Mode.
- حجز تجريبي باستخدام Local Storage.
- لا يوجد Backend ولا يتم حفظ بيانات دفع.
